# FBapi_analysis
# 以Rfacebook進行資料抓取，並進行後續分析

## 現有檔案
FB_getpage
> 抓取特定粉資專頁資料並存檔

FB_textmining
> 針對每則Po文的內容進行文字探勘並畫出文字雲

[KoWJ_analysis](https://github.com/tan800630/FBapi_analysis/blob/master/KoWJ_analysis.md)
> 對柯文哲粉絲專頁進行分析-只使用getpage資料

liker_analysis2
> 將每則po文的按讚者紀錄下來並分類
> 主要分析從以前到現在只按了一次讚的使用者比例(one-shot-liker)
